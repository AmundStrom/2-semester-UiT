You need python and pygame installed to run this program.

In config.py you can change some attributes of different objects.
You can also change how much impact each rule will have on the boid and hoik.
This means that you can set the impact to zero and the rule will no longer be active,
and everything heigher than zero will increase the strength of the rule.

You can also spawn new objects on the position of the mouse, via different keys
	- Press C for circle obstacle
	- Press R for rectangle obstacle
	- Press B for boid
	- Press H for hoik
